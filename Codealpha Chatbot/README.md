# 🤖 Rule-Based Chatbot in Python

A simple Rule-Based Chatbot built using Python. This chatbot can respond to greetings, provide date and time information, detect basic user moods, and includes fun mini-games such as Number Guessing, Quiz, and Rock-Paper-Scissors.

---

## 📌 Features

- 👋 Greeting based on the current time
- 💬 Basic conversation support
- 📅 Display current date
- ⏰ Display current time
- 📆 Display current day
- 😊 Mood detection
- 🎮 Mini Games
  - Number Guessing Game
  - Python Quiz Game
  - Rock Paper Scissors
- 🚪 Exit command
- ⚠️ Exception handling

## 🎮 Available Games

### 🎯 Number Guessing Game

- Guess a random number between 1 and 10.
- Three attempts are provided.

### 🧠 Python Quiz Game

- Five multiple-choice questions.
- Displays the final score.

### ✊ Rock Paper Scissors

- Play against the computer.
- Option to play multiple rounds.
